Option pricer github.com/mathsam/option_analysis

A suites of tools in C++ doing option pricing using Monte Carlo simulation. Includes random number generation, geometric brownian motion generation, linear algebra, least square Monte Carlo classes and etc.

UltraFinance github.com/wjlei1990/UltraFinance

A suites of tools in Python which scrapes stock and option data from internet, maintains a local MySQL database, and provides a set of APIs for easy data access.

Web apps

Web applications in this website are built using the above tools as back end. The front end is built using mainly Flask web-framework and a variety of technologies including javascript, Redis, simplekv, google chart and etc.

Illustration of the architectural designs used in Option Pricer and Volatility Analysis web applications. The C++ library is interfaced with Python by using Ctypes. MySQL database is updated daily by a separate process.
